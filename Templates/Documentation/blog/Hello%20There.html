Hello There

Introduction

Hello There,

Welcome to T-blogs project. T-blogs helps you create traditional blogs with disqus comments integration using Tiddlywiki 5.

It is based on the Node JS version of TW5. Despite that you do not need programming skills to make use of the platform. Essentially it means you can open your favorite text editor, write your blog post and publish it as HTML - without so much as opening your browser even once.

This documentation webpage itself is created using TW5 based on Stripes.js documentation. Apparently eating your own dog food is a thing, who knew!.

A sublime text based workflow is given to help the beginners to TW5 platform.

Introduction

TW5 is an excellent static webpage creater. However, almost everyone using the platform uses it as TW5 itself. While this might be easier and more flexible for day to day uses, for a blog we need to branch out to the static webpage format. The reasons are three

  • It is easier to theme a html from scratch than TW5. TW5 already has built in themes and elements. You need to ensure that in addition to making everything presentable, the customisations are not preventing access to the important parts of TW5. If a person is already familiar with the TW5 interface, adapting to a radically different theme could be difficult too. I assume this is the reason why most of the interface-customisation attempts has been limited to color-palettes or adding a few menubars.
  • The search engine optimisations. To run a blog, much of array of the TW5 parts is unnecessary. It would be unfair to ask the viewer to load all these on top of the blog page, depending on the fanciness of your blog might itself be quite heavy.
  • Plain HTML means people do not require uber-deep knowledge of TW5 platform to use it.
  • Comments. A nice blog demands user engagement. The attempts to successfully introduce commenting by a visitor to TW5 has been going on for quite some time now. The best stuff achieved so far depends on google forms and google sheets, which has its own shortcomings. Disqus comments has been enabled in all the themes under the project.

However the known attempts to tap into such a property of TW5 has been far and few. Jeremy has a blog edition with a few bash scripts in the main site, which is the inspiration for this attempt.
